# Test Cases

## TC-01 맞춤 여행 코스 생성

Related AC: AC-01, AC-06, AC-07

### Given
사용자가 목적지, 일정, 동반자, 여행 컨셉을 모두 입력한다.

### Input
- 목적지: 제주도
- 일정: 2박 3일
- 동반자: 연인
- 여행 컨셉: 힐링

### When
코스 생성 버튼을 클릭한다.

### Expected Result
5초 이내에 3가지 테마별 추천 일정표가 표시된다.

## TC-08 추천 코스 비교

Related AC: AC-11

### Given
3가지 추천 코스가 생성되었다.

### When
사용자가 추천 결과 화면을 확인한다.

### Expected Result
각 코스의 테마, 예상 소요시간, 주요 장소, 추천 이유를 비교할 수 있다.
