# PRD

## 프로젝트 이름

AX 기반 초개인화 여행 큐레이션 플랫폼

## 문제 정의

한 번의 여행을 계획하기 위해 사용자는 평균 11개 이상의 채널을 탐색하고 20시간 이상을 소요한다.

블로그, SNS, 지도, 예약 플랫폼 등에 정보가 흩어져 있어 여행 계획 과정이 오래 걸리고, 광고성 글과 실제 후기를 구분하기 어렵다.

## 목표 사용자

- 시간 효율을 중시하는 2030 직장인
- 제한된 시간 안에 맞춤형 여행 일정을 만들고 싶은 사용자
- 인플루언서의 실제 여행 코스를 참고해 빠르게 일정을 구성하고 싶은 사용자

## 핵심 가치 제안

파편화된 여행 정보와 인플루언서의 여행 팁을 하나의 플랫폼에서 통합하고, 사용자의 목적지, 일정, 동반자, 여행 컨셉을 바탕으로 3가지 테마별 맞춤 여행 일정을 빠르게 제안한다.

## 핵심 시나리오

사용자가 목적지, 일정, 동반자, 여행 컨셉을 입력한다.

코스 생성 버튼을 클릭하면 5초 이내에 3가지 테마별 추천 일정표가 표시된다.

추천 결과에는 주요 장소, 추천 이유, Day 1 / Day 2 형식의 일정표, 지도 기반 동선 영역이 포함된다.

사용자는 인플루언서 추천 코스를 내 일정에 추가할 수 있다.

## MVP 기능 범위

1. 여행 조건 입력
   - 목적지
   - 여행 일정
   - 동반자 유형
   - 여행 컨셉

2. 필수 입력값 검증
   - 목적지가 없으면 안내 메시지 표시
   - 일정이 없으면 안내 메시지 표시
   - 여행 컨셉이 없으면 안내 메시지 표시
   - 필수 항목 누락 시 추천 생성 차단

3. AI 기반 맞춤 여행 코스 추천
   - Mock Data 기반으로 3가지 테마별 추천 코스 생성
   - 실제 AI API 연동은 MVP 이후 확장 기능으로 둔다

4. 추천 일정표 표시
   - 3가지 추천 코스 표시
   - 각 코스는 테마, 설명, 주요 장소, 추천 이유를 포함
   - Day 1, Day 2 형식으로 일정표 표시

5. 지도 기반 동선 시각화
   - 실제 지도 API 대신 MVP용 지도/동선 대체 UI 표시
   - 장소 마커와 이동 순서를 화면에 표시

6. 인플루언서 코스 연동
   - 인플루언서 추천 코스 표시
   - 내 일정에 추가 버튼 클릭 시 추가 상태 표시

## 제외할 기능

- 항공권 직접 결제
- 숙박 직접 결제
- 투어 상품 직접 결제
- 실제 Kakao Map API 연동
- 실제 AI API 호출
- 로그인 / 회원가입
- 유료 결제 기능

## 성공 기준

- 사용자가 목적지, 일정, 여행 컨셉을 입력하면 추천 코스를 생성할 수 있다.
- 필수 입력값이 없으면 안내 메시지가 표시되고 진행이 차단된다.
- 추천 결과는 3가지 테마로 표시된다.
- 추천 결과에는 일정표와 지도/동선 영역이 포함된다.
- 인플루언서 코스를 내 일정에 추가할 수 있다.
- npm.cmd test가 통과한다.
- npm.cmd run build가 성공한다.
