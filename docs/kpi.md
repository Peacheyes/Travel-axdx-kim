# KPI

## Primary KPI

### Recommendation Completion Rate
사용자가 여행 조건을 입력한 뒤 추천 결과까지 도달한 비율.

목표:
- MVP 테스트 기준 70% 이상

---

## Secondary KPI

### Recommendation Save Rate
추천된 코스를 사용자가 저장하거나 내 일정에 추가한 비율.

목표:
- MVP 테스트 기준 30% 이상

---

### Influencer Course Sync Rate
인플루언서 코스를 내 일정에 추가한 비율.

목표:
- MVP 테스트 기준 20% 이상

---

### Recommendation Result Drop-off Rate
추천 결과 화면에 도달한 뒤 아무 행동 없이 이탈한 비율.

목표:
- 낮을수록 좋음

---

## MVP Measurement Events

- recommendation_button_clicked
- recommendation_result_viewed
- recommendation_course_saved
- influencer_course_added
- required_input_missing
